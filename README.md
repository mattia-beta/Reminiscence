 ----------------------------------------
                 SETUP
 ----------------------------------------

 1)  Install all required gems with
     gem bundle install

 2) Setup Facebook API changing APP_ID and APP_SECRET in
    /config/initializers/constants.rb

 3) Run server with
    rails s

 4) Open your favorite browser
    http://127.0.0.1:3000
